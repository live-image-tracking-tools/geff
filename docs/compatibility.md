# Compatible Libraries

The following libraries can read and/or write GEFF files:

- [`motile-tracker`](https://github.com/funkelab/motile_tracker)
- [`traccuracy`](https://github.com/live-image-tracking-tools/traccuracy/pulls)
- [`ultrack`](https://github.com/royerlab/ultrack)
- [`track_gardener`](https://github.com/fjorka/track_gardener)
- [`laptrack`](https://github.com/yfukai/laptrack)
- [`trackastra`](https://github.com/weigertlab/trackastra)
- [`TrackMate`](https://imagej.net/plugins/trackmate/)
- [`InTRACKtive`](https://github.com/royerlab/inTRACKtive)
- [`tracksdata`](https://github.com/royerlab/tracksdata)
- [`napari-geff`](https://github.com/live-image-tracking-tools/napari-geff)
